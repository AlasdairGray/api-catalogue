Individual Employment

Endpoint URL:

  • Sandbox - https://test-api.service.hmrc.gov.uk Production - https://api.service.hmrc.gov.uk

Documentation URL:

  • https://developer.service.hmrc.gov.uk/api-documentation/docs/api/service/individual-employment/1.1

Contact:

  • SDSTeam@hmrc.gov.uk

Description:

This API in particular provides information about an individual’s history of employment for a given tax year, as reported to HMRC by their employer(s) through the PAYE process.
